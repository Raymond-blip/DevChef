# 🍳 DevChef: Recipes for Coders

*A playful recipe manual built for developers, not chefs!*

[![Live Demo](https://img.shields.io/badge/Live%20Demo-GitHub%20Pages-blue)](https://your-username.github.io/devchef)
[![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)](https://opensource.org/licenses/MIT)

## 🎯 What is DevChef?

DevChef transforms cooking into something developers understand - with recipes written like code documentation, ingredients formatted like package.json, and cooking steps that feel like functions. No more vague cooking instructions!

## ✨ Features

### 🤖 AI Recipe Generator
- Generate custom recipes for any food you can imagine
- Smart ingredient recognition and cooking method detection
- Nutritional analysis and energy effects
- Support for dietary restrictions and complexity levels

### 👥 Team Meal Planning
- Create dev teams for collaborative meal planning
- Weekly meal calendar with scheduling
- Team recipe sharing and voting
- Shared shopping lists and coordination

### 🔐 User Authentication
- Personal recipe favorites and cooking history
- Track your cooking progress and achievements
- Personal dashboard with cooking stats
- Gamified cooking experience

### ⏱️ Smart Cooking Timer
- Recipe-specific timer presets
- Multiple timer support with notifications
- Audio alerts and visual progress tracking
- Integration with recipe steps

### 🍽️ Recipe Collection
- Developer-themed recipes with commit messages
- Yoruba/Nigerian cuisine with authentic flavors
- DSA (Data Structures & Algorithms) themed meals
- Nutritional information and energy benefits

## 🚀 Quick Start

### Option 1: View Online
Visit the live demo: [DevChef Live Demo](https://your-username.github.io/devchef)

### Option 2: Run Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/devchef.git
   cd devchef
   ```

2. Open `index.html` in your browser:
   ```bash
   # On Windows
   start index.html
   
   # On macOS
   open index.html
   
   # On Linux
   xdg-open index.html
   ```

3. Or serve with a local server:
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js (if you have http-server installed)
   npx http-server
   ```

1. **Sign Up/Login**: Create an account to access all features

## 🎨 Design Philosophy

### Developer-Centric Design
- **Code-like syntax**: Recipes written like documentation
- **Programming metaphors**: "Push to oven", "Commit your sauce"
- **Debugging tips**: Troubleshoot common cooking mistakes
- **Version control**: "Rollback" if you over-salt

### Inclusive & Fun
- **Cultural diversity**: Yoruba cuisine alongside international dishes
- **Skill levels**: Beginner to Advanced complexity
- **Educational**: Learn about nutrition and cooking science
- **Gamified**: Achievements and progress tracking

## 🤝 Contributing

We love contributions! Here's how you can help:

### Adding Recipes
1. Fork the repository
2. Add your recipe to `recipes.js` following the existing format:
```javascript
{
    title: "Your Recipe Name",
    description: "Brief description with dev humor",
    complexity: "beginner|intermediate|advanced",
    runtime: 15, // minutes
    commit_message: "feat: add awesome recipe for productivity",
    ingredients: [/* array of ingredients */],
    instructions: [/* cooking steps */],
    nutrition: {/* nutritional info */},
    energy_effects: "How this food helps with coding"
}
```

### Improving Features
1. Fork and create a feature branch
2. Make your changes
3. Test thoroughly
4. Submit a pull request with description

### Bug Reports
Open an issue with:
- Steps to reproduce
- Expected vs actual behavior
- Browser and device information
